Heres how to build a new PC!

Have you ever wondered how you can make your own computer? Here's how!

Getting your parts ready.


Before putting your computer together its important to do research on the parts that make up a computer. There are many different parts available online. These are the basic components.

  • Motherboard
  • CPU (Central Processing Unit)
  • RAM (Random Access Memory)
  • GPU (Graphical Processing Unit / If Your CPU does not have integrated graphics.)
  • Storage: SSD (Solid State Drive) or HDD (Hard Disk Drive)
  • PSU (Power Supply Unit)
  • PC Case (This is not necessary if you have a low budget.)
  • Monitor
  • Keyboard and mouse
  • Screwdriver and other various tools

It's important to make sure these parts are compatible with each other. A powerful website that I recommend using when picking out the parts you want is Pc Part Picker. This websites helps with making sure the parts you choose are compatible with each other.

Pc components can get expensive if you choose higher end products, so it is important to set a budget going in to the build. Before you build the computer is it also important to create a boot drive to your preferred operating system. The only thing you need is a USB drive and a working computer. Below is a helpful video on how to download the Windows 10 operating system.


Putting it all together.


Once you have all of your parts, its time to start putting it together. The basic steps go as follows:

  1. Install the CPU in to the motherboard
  2. Install the memory
  3. Install the CPU cooler
  4. Install the motherboard in to the case, plug in case fans and front IO
  5. Plug in or install storage
  6. Install the power supply and plug all the required cables in
  7. Install the graphics card

Building a computer can get confusing very quickly so having a video guide to follow or a person that has done it before is the easiegit st way to ensure that you are doing everything correctly. You do not want to damage any of the parts so be gently and move slowly. If your parts are not plugging in or connecting easily do not keep pushing.

Downloading software and getting it to work.


Once you have finished building your computer and verified that everything is working, you need to install software on to the machine. The easiest way to do this is to take the USB boot drive you previously created and plug it in to a USB port located on the side of the motherboard. If everything worked correctly, when you turn on the PC it should prompt you to start installing Windows, or whatever operating system you are choosing to use.
